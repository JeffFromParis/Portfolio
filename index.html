About Me

Hey there,

My name is Jean-François Vergne, I'm a 0 years old Robotics Engineer. I was born in 1985 in Normandy, France. As far as I remember I have always been attracted by robots. Not necessarily creating robots, but using them, programming them. After a few years travelling a lot across Europe and the USA, commissioning mobile robots for major companies (Procter&Gamble, Volkswagen, BASF, Amazon, Renault, BMW...) I wanted to get new skills and I motivated myslef to learn how to create websites. I realized that I was interested in coding in a large way, not only for robots. I am still currently learning, this page will probably evolve with time as I will get new skills on my way to become a Fullstack Developer.
